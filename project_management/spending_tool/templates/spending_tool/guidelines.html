Guidelines

  • Edit Current Expenses: This information should be updated at the beginning of each quarter. You can view the expense from the previous, current and future quarter.

  • Review Expenses: Once expenses have been entered for the current quarter you are able to review them here.

  • Project Summary: Enter a description for the Project Overview and the Business Value to Cisco. This is to be entered only once at the start of the project, but can be changed throughout by the admin if the project changes.

  • Status: The Project description should be updated quarterly. Each time a status is submitted it will be saved and can be viewed under Previous Statuses.

  • Milestone Schedule: Update the project milestones each quarter with a brief description, due date and the % completion.

  • Project Details:This infomation should be entered at the start of the project, but can edit throughout if any changes need to be made.

  • Attach Documents: In the Documents dropdown list select attach document, so you can upload any funding and project related documents here.

  • Create Report: In the Documents dropdown list select create report which will create an excel document of the expenses for the project.


For the best user experience we recommend using Google Chrome.
